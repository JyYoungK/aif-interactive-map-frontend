## Setting up

##### Clone the repo

```
$ git clone https://github.com/paulowe/aif-interactive-map-frontend.git
$ cd aif-interactive-map-frontend
```

## Running the app
```
$ open ./index.html
```
or simply double-click on ```index.html``` . A static preview should display on your preferred browser.
